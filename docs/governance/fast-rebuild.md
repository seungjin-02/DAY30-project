# Fast Rebuild (증거 문서)

## 목적
이 문서는 이 시스템의 핵심 설계가 설명이나 참고 자료 없이도 즉시 코드로 재구현 가능함을 증명하기 위한 구조적 증거로 활용된다.

이 코드는 정확성이나 완성도를 목표로 하지 않는다.

목표는 설계의 핵심 구조가 사고와 손에 고정되어 있는지를 확인하는 데 있다.

---

## 제약 조건
- 제한 시간: 20~30분
- 코드 길이: 약 30~50줄
- 주석 없음
- 설명 없음
- 외부 참고 자료 없음

---

## 보장되는 구조적 성질
- 상태 흐름은 단방향이다 (STABLE → DEGRADED → HUMAN_REQUIRED)
- HUMAN_REQUIRED는 종착 상태다
- AI는 판단을 수행하지 않는다
- AI는 시스템을 멈출 수만 있다
- 최종 판단과 책임은 인간에게 있다

## 해석 규칙
이 코드를 설명 없이 즉시 재구현할 수 있다면 이 설계는 안정적으로 고정된 것으로 간주한다.

즉시 재구현할 수 없다면 이 설계는 아직 완성되지 않은 상태다.
