# Decision Boundary

## Scope
본 문서는 시스템이 자동으로 수행할 수 있는 행위의 상한과,
그 상한을 초과하는 모든 결정이 인간에게 귀속되는 지점을 정의한다.

## Fixed Boundaries
- 시스템은 가치 판단, 선택, 우선순위 결정을 수행하지 않는다.
- 전이는 상태 값이 아닌, 사전에 정의된 규칙 평가 결과로만 발생한다.
- AI는 결정에 직접 사용되지 않는 정보 정리 및 가시화만 수행한다.
- 결정의 결과에 대한 책임은 언제나 인간에게 귀속된다.

## Prohibited Capabilities
- AI 출력이 전이 트리거로 직접 사용되는 로직
- 규칙 충족 여부와 무관하게 자동 전이를 허용하는 예외 처리
- 전이 결과의 적절성, 성공 여부를 자동으로 판단하는 로직
- 인간의 명시적 입력 없이 결정을 완료하는 모든 흐름

## Enforcement Rules
- 하나의 입력에 대해 복수 규칙이 동시에 상충할 경우 자동 전이를 중단한다.
- 상태 계산 또는 규칙 평가 결과가 신뢰 구간을 벗어날 경우 HUMAN_REQUIRED로 전이한다.
- 사전 정의된 상태 또는 규칙 범주에 속하지 않는 입력이 발생할 경우 판단을 인간에게 위임한다.

## Change Policy
다음 항목 중 하나라도 변경되는 경우,
본 문서는 설계 원칙 변경으로 간주한다.

- 판단 주체의 귀속 (시스템 / AI / 인간)
- 자동화가 허용되는 범위
- 전이를 트리거하는 주체 또는 조건의 성격
- HUMAN_REQUIRED 상태의 발생 조건 또는 해제 조건
