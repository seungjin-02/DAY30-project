# DAY28 — Scenario & Counterexample Mapping

## 오늘 고정한 것
- 전이는 상태가 아니라 규칙 범주로만 설명된다.
- 자동화 중단의 원인은 상태가 아니라 경계 문서에 있다.
- 반복, 성공률, 패턴은 자동화 확장의 근거가 아니다.

## 확인된 침범 경로
- 상태를 자동화 중단의 원인으로 사용하는 언어
- 규칙 결과를 판단 입력으로 재사용하려는 시도
- 인간 판단 패턴을 규칙으로 흡수하려는 유혹

## 오늘 차단한 선언문
- 반복 판단은 자동화 확장의 근거가 아니다.
- 성공률은 판단 이전 자동화를 정당화하지 않는다.
